DQN¶

DQN package for deepQ learning

This module provides methods for applying deep Q learning to atari games, with a focus on the game ‘pong’

This package contains two classes:

Qmemory: for storing and retrieving ‘experience memory’ of an agent, and deepQ: which contains a series of methods for trainingm and testing agents

DQN.Qmemory(N_mem, N_x, N_y, N_frames) memory object for Q learning
DQN.deepQ(game, HYPERPARAMS, PARAMS) Object for deep Q learning, for solving openai gym environments
